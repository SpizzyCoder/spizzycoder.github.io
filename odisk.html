ODisk stands for Overwrite Disk
Unfortunately, "delete" is not enough to safely remove data on a hard disk. If you delete data in the traditional way, this space is only freed in the file system. This means that someone who knows that there was a file on this area (even if it was deleted) can restore it. You don't even need to know where the file was, you can simply scan the entire drive and recover "deleted" files. To recover files, you can use the photorec tool. You can specify the drive from which you want to recover files, specify the destination of the recovered files and start the recovery. To safely remove data, you should overwrite the storage medium with data. There is also a Unix tool called shred for this task. I have programmed ODisk at the beginning of my Rust time. It is a simple tool, but it can safely overwrite data from the hard disk.
