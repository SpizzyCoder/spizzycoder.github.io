Contribute

I also programmed this program at the beginning of my Rust time. I often found myself renaming multiple files via the console. For this there is the Linux tool rename. I wanted to simply replace words or characters with other words or characters. In the past the "rename" tool was a bit complex for me. It needed an expression to rename files. I used to find this an overkill. But today I realized how handy this expression was and still is. It accepts the same expressions as the program vim for example. So you can decide if you want to replace all occurences or only the first one for example. I programmed this tool because I wanted a crossplatform terminal solution. So I can use the same tool on GNU/Linux as well as on Windows.
